Playing instructions

To move a piece, select it by touch. It will be highlighted. Then select the position where it should go. Watch the title for game information such as the state of the game or if it was an illegal move.
Click the Play button to let Android make a move.
Use the navigation buttons or the history bar to navigate through the game history. If you go back and make a move from an earlier point in the game, the game history gets truncated from that point.

Options

Choose a game against Android or two player mode.
If you are in two player mode, you can check to rotate the board automatically after each move.
If you want to view the valid moves of the piece you selected to move, check the 'Show moves'. Set the maximum time Android is allowed to 'think'.

Views

Press the 'more' button to switch controls to one of the following:
Default 'player' view with navigation buttons.
Captured pieces.
Clock. Press the clock icon to set the time.
Show annotations. Click the text area to edit. PGN history.

Save and load

You can save your game to the saved games list via the menu. Choose 'Save game' and fill in the dialog so that you can find it again.
In order to load a saved game, choose 'Load game' from the menu.

Tips

Use the clock and set the time to 2 or 5 minutes. Try to beat Android before time runs out. Don't forget to set the playing level of Android to an appropriate amount (2 seconds)
